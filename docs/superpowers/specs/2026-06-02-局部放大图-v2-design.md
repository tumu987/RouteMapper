# 局部放大镜 v2 设计规范

## 背景

北疆路线中，喀纳斯与禾木相距仅约 0.15°，在全图 10°+ 跨度的主图上几乎重叠。
需要一个**圆形放大镜**来展示该区域的细节。

## 核心决策

| # | 维度 | 决策 |
|---|------|------|
| 1 | 形状 | 圆形放大镜 |
| 2 | 位置 | 选取框附近空白处，螺旋搜索 |
| 3 | 放大镜内容 | 底图+省界+城圈+路线+天次+距离/时间+景点(全部) |
| 4 | 主图内容 | 保留城圈+路线；去掉天次/距离/景点标注 |
| 5 | 放大倍数 | 8×（城圈缩至 1/8，视觉与主图一致） |
| 6 | 放大镜城圈 | `CITY_RADIUS / 8` ≈ 0.016° |
| 7 | 排布逻辑 | 复用 LayoutEngine 完整管道，**跳过全局松弛** |
| 8 | 选取框 | 虚线圆，范围与放大镜展示范围一致 |
| 9 | 连接线 | 无 |
| 10 | 底图 | 与主图一致（`render_base_map`） |
| 11 | 多簇 | 各生成一个放大镜 |
| 12 | 渲染方式 | 独立 figure → RGBA 截图 → 圆形裁剪 → `fig.figimage` 贴入主图 |
| 13 | 经纬线 | 默认密度（cartopy 自动） |
| 14 | 标题 | 不加 |
| 15 | 方/圆比例 | 方图直径 = 圆直径 × 2（圆占方图 50%） |
| 16 | 选取框大小 | 与放大镜实际展示范围完全一致 |
| 17 | 边框 | 红色 3px |
| 18 | 元素重叠 | 方图大一倍，LayoutEngine 有足够空间自动避免 |
| 19 | 外部路线 | 延伸到框外，不画外部城市节点和标注 |
| 20 | 与行程表冲突 | 优先避开（螺旋搜索纳入碰撞检测） |
| 21 | 回退策略 | 缩小 20% 重搜一轮；仍失败则放弃该放大镜 |

## 架构

```
加载配置
  → 主图渲染（与现有一致，但标注跳过 zoom 城市）
  → 检测过近城市对
  → 连通分量分组
  → 计算放大范围（zoom 半径 + render 半径）
  → 螺旋搜索放置位置（纳入行程表碰撞）
  → 独立 figure 渲染放大图（LayoutEngine 完整管道，无松弛）
  → 截图 → RGBA → 圆形裁剪 → fig.figimage 贴入
  → 主图虚线圆选取框
  → 保存
```

## 关键参数

```
ZOOM_FACTOR = 8               # 放大倍数
INSET_THRESHOLD = 0.02        # 触发阈值（对角线比例）
INSET_PADDING = 2.0           # zoom 区域外扩倍率
RENDER_MARGIN = 2.0           # 方图/圆图直径比
```

## 范围计算

```
zoom_radius = max(cluster_half_width * INSET_PADDING, 0.15) * 5 / ZOOM_FACTOR
zoom_radius = max(zoom_radius, cluster_half_width * 1.3)
render_radius = zoom_radius * RENDER_MARGIN
```

- `zoom_radius`：圆形放大镜展示的地理半径
- `render_radius`：方图渲染的地理半径（大一倍）
- 选取框画在 zoom 范围（与放大镜一致）

## 放置

螺旋搜索，从选取框中心出发，逐个检查：
1. 不超出主图范围
2. 不与行程表重叠
3. 不与已有放大镜重叠
4. 不与城市节点重叠

找不到 → 缩小 20% 重试 → 仍失败则放弃。
