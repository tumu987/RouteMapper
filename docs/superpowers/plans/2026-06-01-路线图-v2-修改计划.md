## 一、架构

```
config.py  →  数据加载/验证/默认值/颜色常量
layout.py  →  布局引擎（碰撞检测 + 逐层放置 + 全局松弛）
renderer.py → 渲染引擎（matplotlib绑定 + 跨平台字体）
generate.py → 主入口（编排 + CLI）
```

JSON配置驱动。从零重写，整合nanjiang_v3+generate_final优点。

## 三、放置顺序（最终版）

```
L1 底图+省界（固定）
L2 路线线条（固定，注册碰撞系统）
L3 城市节点（固定坐标）
L4 天次标注     ← 不灵活，先占位
L5 距离/时间    ← 中等灵活
L6 景点标注     ← 最灵活，后填缝
L7 行程表+标题（最后）
```

**原则**：灵活度低的对路线位置要求严 → 先放；灵活度高的可绕圈 → 后放。

## 四、布局算法

### 4.1 三阶段搜索（天次/距离/时间共用）

**阶段1**：中点垂线，法线方向最小距离，两侧递进
**阶段2**：极小滑动（≤0.12°沿路线偏移），仅在垂线堵死时
**阶段3**：远距离回退

### 4.2 核心参数

| 参数 | 值 | 说明 |
|------|-----|------|
| ROUTE_HW | 0.07° | 路线半宽（含光晕） |
| MIN_DIST_EXTRA | 0.10° | 标签中心到路线最小额外间距 |
| 文字尺寸 | `pt × DPI/72` | 精确公式，无经验系数 |
| 中文字宽 | =字高 | 方形 |
| 英文/数字 | =字高×0.6 | |
| CJK检测 | `'一'<=c<='鿿' or '　'<=c<='〿'` | 精确CJK范围 |
| MARGIN_SAME | 0.15° | 同类型间距 |
| MARGIN_DIFF | 0.15° | 不同类型间距 |
| MARGIN_PLACED | 0.06° | 通用放置margin |
| MARGIN_ROUTE | 0.10° | 到其他路线安全距离 |
| MARGIN_LABEL | 0.10° | rect_overlap默认margin |

**路线碰撞跳过自己段**：`route_collision`和`is_position_clear`支持`skip_x1/y1/x2/y2`参数，防止标签被自己的路线段拦截。

### 4.3 景点回避路线中点

城外景点（`place_attraction`）选位时计算到所有路线中点的距离，优先选最远的——给天次留出空间。

## 五、全局松弛系统

### 5.1 流程

所有元素放置完成后，运行`relax_overlaps()`：

1. 扫描所有元素对，检测重叠（margin 0.10°）
2. 收集修复方案：低优先级元素移位
3. 统一应用所有修复（不改动索引）
4. 重复直到无重叠或10轮

### 5.2 优先级

`day(0) > dist/time(1) > attr(2)` — 景点最低，优先被推。

### 5.3 配对保持

`dist_3`和`time_3`作为配对一起移动，保持原有相对位置。

### 5.4 回退腾挪

景点（优先2）移不了时，尝试小幅度移动堵路的day/dist/time（优先0/1）腾空间。位移量限0.12°以内。

### 5.5 搜索圈

`(0.08, 0.16, 0.28, 0.45, 0.70, 1.0, 1.5°) × 8方向`。

## 六、景点系统

### 6.1 城圈内景点

- 折线引线：城圈边缘→斜线→拐点→水平线→文字组
- 每城所有景点共享一条引线
- 仅对角线方向搜索（45°/135°/225°/315°）
- 每城最多2主要+2次要

### 6.2 城外景点

- 坐标点打圆点标记
- 默认无引线，文字紧邻坐标（距离≤0.15°）
- 距离>0.15°时用折线引线
- 8对角线方向搜索，最小距离0.06°

## 七、行程表

- 自动搜索地图空白处，偏好左上角
- 单列放不下时自动双列/拓宽东边界
- 14pt文字，无背景底色

---

## 八、图幅

- 地图范围：城市经纬度±1.0°，向外取整到0.5°
- 北边界自动扩展以容纳行程表
- 长宽比自动适配路线形状（ratio 0.5-2.0）
- 22-26英寸, 120 DPI

---

## 九、命令行

```bash
python generate.py <config.json>     # 生成单条路线
python generate.py --all             # 生成全部路线
python generate.py --list            # 列出可用路线
python generate.py <config.json> -o <output.png>  # 指定输出路径
```
