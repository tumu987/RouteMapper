## 四、布局算法

### 4.1 天次/距离/时间：三阶段搜索

**阶段1（中点垂线）**：法线方向最小距离，两侧递进
**阶段2（极小滑动）**：≤0.12°沿路线偏移，仅在垂线堵死时
**阶段3（远距离回退）**：增大距离倍数

### 4.2 核心参数

| 参数 | 值 | 说明 |
|------|-----|------|
| route_hw | 0.07° | 路线半宽（含光晕） |
| min_dist | route_hw + r_eff + 0.10 | 标签中心到路线中心最小距离 |
| 文字尺寸 | `pt×DPI/72` | 精确公式，无经验系数 |
| 中文字宽 | =字高 | 方形 |
| 英文/数字 | =字高×0.6 | |
| 路线碰撞margin | 0.10 | 到其他路线的安全距离 |
| 标签碰撞margin | 0.02 | 标签间安全距离 |
| 跳过自己路线 | ✓ | 避免标签被自己段拦截 |

### 4.3 天次vs距离/时间

天次对面优先，对面空间不够则同侧可接受。

### 4.4 往返路线

往返段仅注册一次路线，天次分两侧标注，距离/时间只标一次。

## 五、景点系统

### 5.1 城圈内景点

- 折线引线连接城圈：城圈边缘→斜线→拐点→水平线→文字
- 每城所有景点共享一条引线
- 仅对角线方向搜索（45°/135°/225°/315°）
- 紧贴城圈（gap 0.06°）

### 5.2 城外景点

- 坐标点打圆点标记
- 默认无引线，文字紧邻坐标
- 距离>0.15°时用折线引线：点→斜线→拐点→水平线→文字
- 对角线方向搜索，最小距离0.06°
- 全面回退搜索（8角度×4距离）

## 六、行程表

- 自动搜索地图空白处
- 单列放不下时自动双列/拓宽东边界
- 14pt文字

---

## 七、图幅

- 地图范围：城市经纬度±1.0°，向外取整到0.5°
- 北边界自动扩展以容纳行程表
- 长宽比自动适配路线形状（ratio 0.5-2.0）
- 22-26英寸, 120 DPI
